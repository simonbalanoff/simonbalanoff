## 🚀 Featured Projects

#### 📱 [SigEpRush](https://github.com/simonbalanoff/SigEpRush-App) · Swift + TypeScript · [`App`](https://github.com/simonbalanoff/SigEpRush-App) [`API`](https://github.com/simonbalanoff/SigEpRush-API)
Full-stack recruitment management platform built for a real organization — iOS app in Swift backed by a Node.js/Express/MongoDB REST API. Supports 123 registered users with JWT auth, candidate scoring, and real-time updates. Replaced manual spreadsheet workflows and was adopted exclusively by executive leadership.

#### 🩺 [Resident Feedback](https://github.com/simonbalanoff/ResidentFeedback-App) · Swift + TypeScript · [`App`](https://github.com/simonbalanoff/ResidentFeedback-App) [`API`](https://github.com/simonbalanoff/ResidentFeedback-API)
iOS + web platform for surgeons to record and track resident performance. Features biometric auth, secure Keychain storage, role-based access, and a TypeScript/Express/MongoDB backend. Purpose-built for a clinical workflow.

#### 🔍 [UML Visualizer](https://github.com/simonbalanoff/UMLVisualizer) · Java
Parses Java source files and automatically generates interactive UML class diagrams. Built from scratch with a custom parser and graph renderer, no external diagram libraries.

#### 🕸️ Nodera *(private)* · TypeScript
Graph-driven tool for designing and visualizing API and database architectures. Represents endpoints, models, and relationships as an interactive node graph.
